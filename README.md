## Проект "Вычислитель отличий"
___

### Требования

* Linux, Macos, WSL
* PHP >=8.2
* Xdebug
* Make
* Git

### Установка зависимостей

* make install

### Запуск программы

* make gendiff

Пример сравнения двух плоских JSON-файлов [ASCIINEMA](https://asciinema.org/a/rngphDkJ5tb199UTIk71aVNEu)

### Тестирование и проверка

* make validate
* make lint
